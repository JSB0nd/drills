## Чтобы использовать упражнения
### Установите
1. Скачайте код на компьютер.
2. ```cd drills```
3. ```cd server``` ```npm i```
4. ```cd ..```
5. ```cd front``` ```npm i```

### Для выполнения упражнений
Нужно запустить приложение. Упражнения собраны в виде приложения. 
1. ```cd server``` ```node index```
2. ```cd ..``` ```cd front``` ```npm start```
